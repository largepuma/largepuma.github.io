Again, I'm free these days. So, I begin to study mac, for lots of my friends recommend it for a long time. Besides, I like apple's design, and I wish to have a opportunity to learn it.
<!-- more -->
First, we should install mac os x. For I don't own a mac computer, I decided to install it through virtual machine--virtualbox. Follow [a hacker's instruction](http://tek411.blogspot.com/2010/06/osx-in-virtual-box-hackintoshed-vm.html), I installed it successfully. Mac os x is based on unix, and is combination of unix and aqua interface. Ubuntu is distribution of linux, a clone of unix. So they are similiar in many things. The difference is the user interface. Actually, I felt that Ubuntu Unity is similar to mac os x's interface. Both have menu bar, main window and a dock. The difference is ubuntu's dock is in the left, and mac's dock is in the bottom. They are user-focused, and are perfect productive tools.
