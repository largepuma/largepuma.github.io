Lately I am studying python.
<!-- more -->
Lots of people say that python is great language, because using it can largely improve your development efficiency. And as a fact,  many open source software are developed using python, from website to desktop application, and useful tools. First, I study through the [Python Tutorial](http://docs.python.org/tutorial/) and [Dive Into Python](http://diveintopython.org/). Through these materials, I learned the basis of the language and its library, and can write some interesting programs. Second, I think it's essential to study some web framework and some desktop apis. [Django](https://www.djangoproject.com/) is good choice. Using it you can build a customized website quickly. I also plan to study [ PyGObject](http://developer.gnome.org/pygobject/stable/) some time later. Python is not only another language, it's a large community. Join it , study the language, been a  Happy Pythoner.
